# SPA ANGULAR

This project is a single page application using Angular and Angular Material.

## Features

The fitness calculator is a basic method to control your daily diet (protein, carbo and calories).

## Code

To use the application it's necessary to install Material and RxJs.

The project is not 100% done, so i'm still working on some features, as the inputs and outputs to show the daily diet on the Total Screen, and also a save button
to register your weekly and monthly resume.
